# Import Mining

## What? 

It's a tool that collect coupling information based on **explicit** import data.

## Why?

Because you should strive for the least coupling in your classes you possibly can.

## How?

It's written in Kotlin using the libs:
 * Javaparser
 * JGit
 * Weka
